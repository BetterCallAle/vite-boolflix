# Boolflix

This webapp is created by [BetterCallAle](https://github.com/BetterCallAle) for search movies and series form  [The Movie DB API](https://developers.themoviedb.org/3/getting-started/introduction).

## Used library and frameworks:

- [Vue + Vite](https://vuejs.org/)
- [Axios](https://axios-http.com/docs/intro)
- [Bootstrap](https://getbootstrap.com/)
- [Sass](https://sass-lang.com/)
- [FontAwesome](https://fontawesome.com/)
